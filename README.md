# 리액트 앱 실행하기

1. nodejs 설치
2. `npm create vite@latest 프로젝트폴더명`
3. `cd 프로젝트폴더명` -> `npm install` -> `npm run dev`

## 깃허브에서 클론한 리액트 프로젝트인 경우
1. `git clone`
2. `npm intall` <- 최초 1번
- 서버 끌때는 터미널에서 Ctrl + C, Ctrl + Z

# 추가 라이브러리 설치
1. styled-components: 동적 CSS를 처리해주는 라이브러리
- `$ npm install styled-components`
